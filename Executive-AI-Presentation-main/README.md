# Executive AI Presentation Generator

An AI-powered tool for automatically generating professional presentations for executives. This application allows users to input their presentation requirements and automatically generates slides with appropriate content, narration, and visualization.

## Features

- **AI-Powered Slide Generation**: Create presentation slides with minimal input
- **Narration Generation**: Auto-generate speech content for each slide
- **Boardroom Discussion Simulation**: Simulate executive discussions about your presentation
- **Slide Builder**: Customize and fine-tune individual slides
- **Preview Mode**: Review the generated presentation before downloading
- **Multiple Export Options**: Download your presentation in various formats

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- OpenAI API

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   # or
   pnpm install
   ```
3. Run the development server:
   ```
   npm run dev
   # or
   pnpm dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
